# Publishing to GitHub

This repository is configured to publish to the public GitHub remote:

```
git remote add origin https://github.com/petrsramekw/Engineering_holography.git
```

To push the current branch (`work`) upstream, run:

```
git push -u origin work
```

Authentication is required for pushes; supply valid GitHub credentials or a personal access token when prompted. No credentials are stored in this repository or environment.
